### Quick start

install <u>**Imagine**</u> package into your application

```bash

cd yourApp

npm install --save Imagine
```

### Start using <u>`RichText` </u> component

<br />
<br />

##### Example:

```javascript
import { ImagineProvider, Container, RichText } from "imagine";

function App() {
  return (
    <ImagineProvider>
      <Container
        flexDirection={"col"}
        justifyContent={"center"}
        alignItems={"center"}
      >
        <RichText strHTML={'<p>זה טקסט<a href="https://google.co.il">קליק על הלינק</a><strong>זה טקסט מוגדש</strong></p>'}/>
      </Container>
    </ImagineProvider>
  );
}
```
