### Quick start

install <u>**Imagine**</u> package into your application

```bash

cd yourApp

npm install --save Imagine
```

### Start using <u>`Icon` </u> component

<br />
<br />

##### Example:

```javascript
import { Icon } from "imagine";

function App() {
  return (
     <Icon 
           name={'icon'}
           size={'L'}
           color={'#ccc'}
     />       
  );
}
```
