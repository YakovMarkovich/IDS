### Quick start

install <u>**Imagine**</u> package into your application

```bash

cd yourApp

npm install --save Imagine
```

### Start using <u>`Text` </u> component

<br />
<br />

##### Example:

```javascript
import { ImagineProvider, Text} from "imagine";

function App() {
  return (
    <ImagineProvider>
        <Text
          text='Google'
          type='p'
        />
    </ImagineProvider>
  );
}
```
